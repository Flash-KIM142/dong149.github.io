DP문제로, 결국에는 피보나치 수열을 계산하는 문제이다. 간단한 문제지만, 코드를 깔끔하게 짜는 법을 익혔다.

단순한 것이였는데, for문에서 애초에 범위에 해당하지 않으면 실행하지 않고 넘어가는 것을 오늘 처음알았다.. 기초적인 것을 너무 간과했던 것 같다. 정신차리길 바란다.


```c++
#include <iostream>
using namespace std;

int n;
int res;
int dp[1000001] = {0};
int main()
{
    cin >> n;
    dp[1] = 1;
    dp[2] = 2;
    for (int i = 3; i <= n; i++)
    {
        dp[i] = (dp[i - 1] + dp[i - 2]) % 15746;
    }
    cout << dp[n] << endl;
    return 0;
}
```
